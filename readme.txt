■使い方

[Board settings] -> [Board textures] -> [Custom Board] でフォルダを指定して再ログインして下さい

・適用には再ログインが必要な箇所とそうでないところがあります
　ベアオフトレイ背景とダイス変更には再ログインが要ります（たぶん）
　何かしらが変更されない場合は再ログインして下さい

・ダイスの変更は一度で成功しない事があるかもしれません（一部の目だけ変わらない）
　その場合は再度フォルダ指定＆再ログインを繰り返してみて下さい

・チェッカーボーダーを無し、または有りに指定・固定している物があります
　ボーダーをサイト側の設定で自由に変更したい場合は、custom.css の5行目・6行目↓を消して下さい
　------------------------------------          -------------------------------------------------
　.blackpiece,                          或いは  .blackpiece {border: 2px solid #fff !important;}   みたいなやつ
　.whitepiece {border: 0 !important;}           .whitepiece {border: 2px solid #000 !important;}
　------------------------------------　        -------------------------------------------------

■Custom Board リセット

[Board textures] 右上のゴミ箱アイコンを押して再ログインして下さい

・カスタムボード使用後にボードデザインを変更する場合は、基本的に一度このリセットをかける必要があります
